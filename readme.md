🧪 ACID Phone OSINT Tool
📋 Описание

ACID — это легковесный OSINT-инструмент для базового анализа телефонных номеров. Программа представляет собой минималистичное решение с одним предустановленным сервисом, предназначенное для демонстрационных целей и локального использования.

    ⚠️ Обратите внимание: Инструмент не использует коммерческие или премиальные API и имеет ограниченную функциональность. Он создан как основа для дальнейшего расширения и кастомизации.

🚀 Возможности

    🔍 Базовый анализ российских телефонных номеров

    ⚡ Асинхронная архитектура для эффективной обработки запросов

    🧩 Модульная структура — легкое добавление новых источников данных

    🎨 Красочный вывод в консоль (с использованием Rich)

    📱 Поддержка номеров в международном формате

🛠️ Использование
bash

# Показать справку
python acid.py -h
python acid.py --help

# Выполнить поиск по номеру
python acid.py -s 79000000000
python acid.py -search 79000000000

🔧 Расширение функциональности

Программа спроектирована с учетом возможности легкого расширения. Вы можете добавить собственные источники данных, следуя простому принципу:

    Откройте файл acid.py

    Найдите список sites в функции settigs_search()

    Добавьте URL вашего API в формате:

python

sites = [
    "http://num.voxlink.ru/get/?num=",  # Существующий источник
    "https://your-api.com/phone/",       # Ваш источник
    "https://another-service.net/?number=" # Еще один источник
]

Важно: URL должен заканчиваться так, чтобы номер подставлялся в конец строки (например, https://example.com/api/ + 79000000000).

⚡ Асинхронность

Асинхронная архитектура (asyncio + aiohttp) позволяет:

    Одновременно опрашивать несколько источников

    Эффективно использовать время ожидания ответов от серверов

    Легко масштабировать инструмент при добавлении новых сервисов
